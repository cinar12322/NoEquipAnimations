# No Equip Animation

This mod disables the equip animation for weapons in ULTRAKILL, allowing for instantaneous weapon switching.

## Usage
Press **K** to toggle the mod on or off. When toggled, an indicator will appear on the HUD:
- **Lime Square (■)**: Mod is enabled (No equip animation).
- **Red Square (■)**: Mod is disabled (Default behavior).

## Installation
Use a mod manager like r2modman or Gale, or install manually by placing `NoEquipAnimation.dll` inside your `BepInEx/plugins` folder.
