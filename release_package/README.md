# No Equip Animations

Instantly skips drawing and equip animations for all weapons and arms/fists in **ULTRAKILL**.

ULTRAKILL'deki tüm silah ve kol kuşanma (equip/draw) animasyonlarını anında geçerek silahlarınızı anında kullanıma hazır hale getirir.

---

## Features / Özellikler

* ⚡ **Instant Weapon Swapping / Anında Silah Değişimi**: Completely skips the draw animation by advancing the animator directly to the idle state in 2 frames.
* 🛠️ **In-Game Toggle / Oyun İçi Aç/Kapat**: Toggle the mod on and off at any time by pressing **`H`** (customizable).
* ⚙️ **PluginConfigurator Integration / Menü Entegrasyonu**: Configurable directly via the in-game options menu if you have PluginConfigurator installed.
* 🎨 **Visual Alerts / Görsel Bildirimler**: Displays elegant, color-coded HUD notifications when toggled.
  * `Equip Animasyonu: AÇIK` (Green / Yeşil) - Animations are enabled (vanilla behavior).
  * `Equip Animasyonu: KAPALI` (Red / Kırmızı) - Animations are disabled (instant switch active).

---

## Installation / Kurulum

### Via Mod Manager (Recommended)
1. Install BepInEx and search for **No Equip Animations** in r2modman / Thunderstore / Gale.
2. Click **Install**.

### Manual Installation / Manuel Kurulum
1. Extract `NoEquipAnimation.dll` from the zip file.
2. Paste the DLL into your game's plugins folder:
   `ULTRAKILL/BepInEx/plugins/`
3. Launch the game and enjoy!

---

## Controls / Kontroller

* **`H`** (Default): Toggle equip animations on/off dynamically.
